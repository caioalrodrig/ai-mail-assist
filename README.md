# AutoU Email Assistant

API for email classification and automatic response generation using AI.

## Production

Frontend: https://autou-eta.vercel.app

To test:
1. Access https://autou-eta.vercel.app
2. Upload `input-sample.pdf`
3. Click "Analisar email com IA"
4. The app will classify the email as productive/unproductive and generate a suggested response

For local development, see [back-end/README.md](back-end/README.md).
